============
Installation
============
 

To run this application, you will have to have Node.js and MongoDB installed on your machine:

    • node.js
    • mongoDB
     

Clone the repo from GitHub:

    • git clone
 

Open the folder:

  
    • cd purdue-ironhacks
  

Install the dependencies:

  
    • npm install
  

Start MongoDB:


    • mongod or sudo mongod
